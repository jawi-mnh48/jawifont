DOCUMENTATION
Awami Nastaliq
========================

This file describes the documentation files included with the Awami Nastaliq 
font family. This information should be distributed along with the 
Awami Nastaliq fonts and any derivative works.

Primary documentation for the Awami Nastaliq font family is on the Awami Nastaliq 
website (http://software.sil.org/awami/).

The AwamiNastaliq-features.pdf file illustrates the use of Graphite features.

The AwamiNastaliq-features.odt file illustrates the use of Graphite features 
(source file for the pdf).

The AwamiNastaliq-typesample.pdf file gives a sample of all the characters in 
the font.

The AwamiNastaliq-typesample.odt file gives a sample of all the characters in 
the font (source file for the pdf).
